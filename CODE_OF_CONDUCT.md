# Code Of Conduct

This project follows Apple's community expectations so that the repository
stays aligned with the upstream ecosystem around
[`apple/container`](https://github.com/apple/container).

Contributors are expected to follow the
[Apple Code of Conduct](https://github.com/apple/.github/blob/main/CODE_OF_CONDUCT.md)
in issues, pull requests, discussions, reviews, and related community spaces.

Please keep conversations respectful, focused on the work, and welcoming to
new contributors.

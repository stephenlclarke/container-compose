## Motivation And Context

Downstream packages can resolve a newer SwiftLog than the repository lockfile.
SwiftLog 1.13.2 makes `log(event:)` the primary handler entry point, so relying
on the compatibility bridge emits warnings for all three `ContainerLog`
handlers.

## What Changed

- Update the SwiftLog lockfile to 1.13.2.
- Implement `log(event:)` for file, OS, and stderr handlers.
- Keep the compatibility entry point.
- Share formatting and metadata merging through one private emission path.
- Test all three primary event paths and metadata precedence.

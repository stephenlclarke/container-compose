## Motivation And Context

Builds without an injected `GIT_COMMIT` report `unspeci` because version output
shortens the complete `unspecified` placeholder as though it were a hash.

## What Changed

- Centralize commit display formatting in `ReleaseVersion.displayCommit(_:)`.
- Preserve missing, empty, and `unspecified` values.
- Keep seven-character display for real hashes.
- Cover nil, empty, placeholder, and real-hash values.
